# Todo List

In this example we create a simple todo list.
The goal is to fully integrate it with a QtQuick UI.

## Build the project

```
$ mkdir build
$ cd build
$ cmake -DCMAKE_PREFIX_PATH=<path/to/Qt6> ..
$ cmake --build .
```

## Run

```
$ ./apptodolist
```
